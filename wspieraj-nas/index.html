Wspieraj Code Club World

Mocno wierzymy w to, że każde dziecko powinno mieć możliwość nauki programowania bez względu na sytuację finansową, dlatego Code Club World jest darmowe i dostępne dla każdego.

Code Club World to organizacja non-profit utrzymywana dzięki darowiznom. Środki finansowe, które otrzymujemy przeznaczone są na tworzenie nowych materiałów dydaktycznych dla wolontariuszy i budowanie zespołu, który wspiera tworzenie lokalnych klubów kodowania na całym świecie, w tym w Polsce.
