Code Club Polska

Kluby kodowania dla dzieci w wieku 9-11 lat
prowadzone przez wolontariuszy

Jak to działa?

Wolontariusz + Miejsce + Projekty Code Club + Dzieci = Code Club

Wolontariusz, który umie programować idzie do miejscowej szkoły podstawowej zabiera ze sobą specjalnie przygotowane materiały, aby uczyć programowania dzieci w wieku 9-11 lat. Inne źródła

Nasza misja

Misją Code Club Polska jest umożliwienie każdemu dziecku w Polsce nauki programowania poprzez udostęnianie materiałów do nauki i stworzenie struktur wolontariatu, które wspierają prowadzenie pozalekcyjnych klubów kodowania.

Zarejestruj swój Klub Kodowania

Aby zarejestrować swój klub kodowania przejdź na stronę Code Club World lub napisz do nas na adres codeclubpolska@gmail.com
